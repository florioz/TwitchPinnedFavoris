# Twitch Favorites Sidebar v0.6.2

## English

This patch fixes the channel favorite button placement on Twitch channel pages.

### Fixes

- Fixed the favorite star button position on Twitch's newer channel header layout.
- Prevented the star button from being inserted inside Twitch's `Follow` button group.
- Removed the visual purple merge between Twitch's `Follow` button and the extension favorite button.
- Improved channel action button detection for `Follow`, `Subscribe`, gift subscriptions, and Bits buttons.

## Francais

Ce patch corrige le placement du bouton favori sur les pages chaine Twitch.

### Corrections

- Correction de la position du bouton etoile avec le nouveau layout des pages chaine Twitch.
- Le bouton etoile n'est plus injecte dans le meme groupe que le bouton `Suivre`.
- Suppression de la fusion visuelle violette entre `Suivre` et le bouton favori de l'extension.
- Meilleure detection des boutons d'action de chaine : `Suivre`, `S'abonner`, abonnements cadeaux et Bits.
