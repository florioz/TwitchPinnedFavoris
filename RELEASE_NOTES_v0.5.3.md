# Twitch Favorites Sidebar v0.5.3

## Mobile

- Suppression des elements de demonstration dans l'app mobile.
- Ajout d'une vraie gestion des profils favoris :
  - changer de profil,
  - creer un profil vide,
  - renommer le profil actif,
  - supprimer un profil sauf le dernier.
- Nettoyage des caches lives/VODs lors d'un changement ou d'une suppression de profil.

## Verification

- `npm run check`
